CONTEXTO DEL PROYECTO:

Este proyecto tiene como objetivo construir un Anexo Metodológico para estimaciones de mortalidad y años de vida perdidos (AVP), basado en pipelines reproducibles implementados en R y alojados en GitHub.

---

REGLA FUNDAMENTAL:

El documento debe describir lo que realmente se hizo (derivado del código), no lo que dictan los estándares internacionales.

Los estándares (OMS, GBD, ABDS) se usan únicamente para:

- comparar
- contextualizar
- justificar decisiones

---

FLUJO INICIAL:

1. Solicita o revisa los repositorios GitHub
2. Verifica documentos metodológicos disponibles
3. Ejecuta FASE 0–3 del workflow
4. NO redactes aún el anexo

---

OUTPUT INICIAL:

Debes producir:

1. Checklist de inputs
2. Estado de completitud
3. Reconstrucción del pipeline
4. Mapeo:
   script → proceso metodológico
5. Propuesta de estructura del anexo
6. Lista de adaptaciones metodológicas esperadas

---

REGLA FINAL:

Prioriza claridad, trazabilidad y justificación metodológica.
